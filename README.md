# oli-m1-thesis

Repository for Oliver Cutbill's M1 thesis on Nesterov Accelerated Gradient Descent based on [this paper](https://arxiv.org/abs/2510.23513) by U. Jang and E. K. Ryu, supervised by Professor [E. Pauwels](https://edouardpauwels.fr/).

# How to compile

1. Install `typst`: [https://github.com/typst/typst](https://github.com/typst/typst)
2. Navigate to repository directory locally
3. Run `typst compile thesis.typ -f pdf`
